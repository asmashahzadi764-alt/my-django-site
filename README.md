# 🍽️ Restaurant Website – Django Project

A modern and responsive Restaurant Website developed using Django, HTML, CSS, and JavaScript. The website provides an attractive user interface where visitors can explore the restaurant, view menu items, learn about services, read customer reviews, and contact the restaurant.

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone <repository-link>
cd restaurant-website
```

### 2️⃣ Create Virtual Environment

```bash
python -m venv env
```

Activate environment:

Windows:
```bash
env\Scripts\activate
```

Linux/Mac:
```bash
source env/bin/activate
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run Database Migrations

```bash
python manage.py migrate
```

### 5️⃣ Start Development Server

```bash
python manage.py runserver
```

Open your browser and visit:

```
http://127.0.0.1:8000
```
